IDEA用户手册摘要

Intellij IDEA的用户手册摘要。更多详细内容请参考IDEA的主页

IDEA功能简介

  • 创建项目文件时可以直接输入包名
  • Run | Configure中配置项目
  • 项目的顶层配置在File | Project Structure中。依次可以配置项目、使用的模块、使用的库、facets（用于配置框架）、Artifacts（设置项目输出的可执行文件）
  • 查找命令对应的快捷键 ⇧⌘A，快捷查找⇧⇧，查找设置项Preferences左上的搜索栏

IDEA常用快捷键

代码补完

  • 基本补完 ⌃Space
  • 智能补完(更精确，⌃⇧Space)
  • 语句补完(自动增加括号和格式) ⇧⌘⏎
  • 查看方法或构造器的参数 ⌘P
  • IDE给出建议，用灯泡显示，可以使用⌥⏎查看详情

界面切换

  • Project ⌘1
  • Run ⌘4
  • Debug ⌘5
  • Terminal ⌥ F12
  • Editor Esc
  • 切换到导航栏 ⌥↖(⌥ Fn+←)
  • 在完整界面和纯代码模式间切换 ⇧⌘F12
  • 切换界面 ⌥F1

编辑器基本操作

  • 添加语句块（{},(),if/else等）⌥⌘T
  • 高亮符号 ⌘F7
  • 查找所有符号引用 ⌥⌘F7
  • 显示文件结构 ⌘F12
  • 插入模板 ⌘J
  • 把代码放入模板 ⌥⌘J
  • 列出可覆盖/实现的方法 ⌘O
  • 复制当前代码 ⌘D
  • 展开/折叠代码 ⌘+ /⌘-

代码风格

  • Reformat code ⌥⌘L
  • Auto-indent lines ⌥⌘I
  • Optimize imports ⌥⌘O

  • 可以在注释中加入TODO，然后在TODO窗口中查看
