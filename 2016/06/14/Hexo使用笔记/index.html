Hexo使用笔记

  1. 基本操作
  2. 自定义操作

总结自己用到的Hexo的实用功能。Hexo的完整文档见官方网站。Hero使用Markdown语法说明见Markdown

基本操作

  • 引用块
    { % blockquote [author[, source]] [link] [source_link_title] %}
    content
    { % endblockquote %}

  • 代码块
    { % codeblock [title] [lang:language] [url] [link text] %}
    code snippet
    { % endcodeblock %}

    或者

    ``` [language] [title] [url] [link text] code snippet ```

自定义操作

  • <span class="question">要做lint??</span> 用左边的标志表示疑问
  • <%- url_for(path) %> 在路径前加上根路径
  • <%- relative_url(from, to) %> 取得与from相对的to路径
  • <%- link_to(path, [text], [options]) %> 插入链接
  • <%- mail_to(path, [text], [options]) %> 插入邮箱
