Concept

Color

  • Using Spectrum Distribution Function to describe color
  • 光谱分布函数比较复杂，而且与颜色是多对一的关系。光本身不等同于颜色！
  • 同色异谱：在特定光照下同色，换成其它光照后异色
    SDF(A) != SDF(B) //异谱
    但是 SDF(A)*Filter(X) == SDF(B)*Filter(X) //在特定光照下同色，色可以认为是谱通过了一个反映光照、反射、吸收情况的滤波器的结果？
  • RGB Color Space：用RGB表示颜色，三个通道独立调整。
  • 选择RGB是因为人类视觉对这三个颜色敏感，但是有些颜色无法用RGB表示（R为负值的颜色）
  • CMY Color Space: Cyan(青)，Magenta(洋红)，Yellow（黄）
  • CMY与RGB互补的减色体系：
    RGB：黑+色 -> 白
    CMY：白-色 -> 黑
  • 印刷时难以得到纯正的黑，所以在CMY上加上纯黑(K),就是CMYK颜色体系
  • HSV Color Space: Cone-shape color space, Hue(色调)，Saturation(饱和度)，Value(亮度)
  • 色调是锥底的角度，饱和度是到中轴的距离，亮度是到顶点的距离
  • CIE(International Commission on Illumination) Color Space:人类所有可见光体系
  • Lab:Lightness(明度)，a(色彩分量：绿->红) b(色彩分量：蓝->黄)

Image & Pixel

  • 图像就是离散二维函数F(xd,yd),每一个离散点就是像素

Triangle Mesh

  • 图形学就是处理3d场景转化成2d的问题
  • 简单几何体的数据结构，用参数表示（球，立方体等）
  • 复杂几何体用三角网格表示，或者参数化曲线/曲面
  • Mesh表示法：三角面列表，顶点列表，三角面与顶点的关系列表
  • Mesh的面法线：表示面的正向；
  • Mesh的点法线：
    • 平均插值法：avg(sum(点所在面法线))
    • 面积加权：sum(点所在面法线*面面积)/（所在面总面积）
  • 渲染：线框，平直着色，光照

Lighting Models

  • 局部光照(直接光照)：物体直接被光照的效果
  • 全局光照：阴影，非光源光照效果(如：折射，反射)
  • 光照能量守恒：I入射 = I漫反射+I镜面反射+I吸收+I折射
  • 光的单位：
    • 立体角𝜔 ds/(r^2)，整个球面是4𝜋
    • 辉度E,Irradiance: Energe/time/area
    • 发光强度I,Radiance:E/𝜔
  • Phong:局部光模型
    漫射+高光+环境光：I = Id + Is + Ia
    漫射：Id = Ii Kd (L⋅N) //Kd 漫反射系数。漫反射只与入射角相关
    高光：Is = Ii Ks (R⋅V)^n //Ks 高光系数；n 高光光亮系数。高光与观察方向相关。
    环境光：Ia = Ii * Ka //Ka 环境光系数；环境光各向完全均匀
  • Shading: Phong高光采用线性插值，可能在亮暗交界处出现马赫带现象。网格面使用平直着色，也没有平滑过渡。需要采用更复杂的插值算法:
    • Gourand Shading：计算各个顶点的强度值，再根据强度值进行插值
    • Phong Shading：计算各个顶点的法线，再根据法线进行插值，再根据插值得到的法线计算强度
