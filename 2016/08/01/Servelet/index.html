HTML HTTP TCP/IP

  • HTML HyperText Markup Language，超文本标记语言
  • HTTP HyperText Transfer Protocol，超文本传输协议
  • TCP/IP 待补充

HTML常用标记

HTTP

  • GET URI?参数
  • POST URI 参数在请求体内
  • 响应：Content-Type: MIME类型
    URL: 协议+服务器+端口+路径+资源?xxx&xxx&xxx…
  • Web服务器应用
    apache - htdocs (index.html, JIT页面？？)
  • tomcat目录：？？
  • JSP就相当于java in html

Web容器

  • 管理Servelet
  • apache <-> tomcat -> servelet
    apache负责socket和session
    tomcat负责servelet生命周期，多线程，xml配置，jsp支持
    tomcat向servelet分发HttpServletResponse/HttpServletRequest
    Servlet.service()处理http响应
  • 处理类继承servlet,extends HttpServlet,并实现doGet doPost，写响应response.getWriter()

MVC

  • JSP(View) <- Servlet(Controller) -> 处理数据库的独立类(Model)
    JSP部署后要重启？？
  • Servlet：创建模型；创建分发->发送JSP
  • Servlet生命周期：初始化->循环service()->destroy
    • init:覆盖，创建数据库链接
    • service:不覆盖， doGet, doPost
      对每个请求，容器创建一个线程到Servlet,不会产生多个servlet实例？

企业网站分层

防火墙->负载平衡->[web层]->负载平衡->[业务层]->DB

Spring

  • Core:Bean Factory(IOC,DI)
  • AOP:面向方向编程
  • DAO:JDBC
  • Context: 上下文？？，国际化资源等
  • ORM: JDO, hibernate, iBatis
  • Web: 请求和绑定
  • MVC: Model2实现,JSP中嵌入的java bean和代码放到控制器当中，JSP被纯化成独立的View
