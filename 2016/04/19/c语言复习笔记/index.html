C语言复习笔记

  1. 《C程序设计语言》
  2. 《C专家编程》

对几本C语言的书进行摘要，回顾C语言中的难点

《C程序设计语言》

  • printf语句中%%表示%
  • extern表示外部变量
  • 定义(definition)用来分配存储空间，声明(declaration)用来指明属性
  • \0的数字值是0
  • 外部变量/静态变量若未初始化，自动初始化为0
    自动变量若未初始化，状态为undef
  • const表示不变的常值变量
  • 取模运算%对浮点数无效
  • && ||不具备传导性
  • 运算符优先级
    1. [] () . ->（成员操作）
    2. ! ~ ++ -- - + * & sizeof(一元操作符)
    3. * / % （算术操作符）
    4. + - （算术操作符）
    5. << >> (移位操作符)
    6. < <= >= > (关系操作符)
    7. == != （关系操作符）
    8. & （位运算）
    9. ^ （位运算）
      10 | （位运算）
    10. && || （逻辑操作符）
    11. ?:
    12. = += *= … (赋值)
    13. , （逗号运算符）
  • 外部变量或外部函数的作用域, 从声明处到声明所在文件末尾
  • 函数体外部的static 仅在本文件内可见
    函数体内部的static 该函数的静态变量(不可重入变量)
  • 无法对register变量执行取地址操作，无论该变量是否真的存在寄存器中
  • 数组初始化后，后面未赋值的元素都初始化为0
  • 指针是变量，数组是常指针
  • 多维数组要指定前N-1维（右面N-1维）的尺度
  • argv[0]是程序名
  • struct默认通过值传递，可以用取地址符号来获得其指针来进行引用传递
  • struct可能会因为内存字对齐而产生空穴
  • typedef int xxx; typedef的基本用法，注意加引号

  • union只能按第一个成员的类型初始化
    union可以用来做对齐：

    union{
        XXX;
        long align_sec;}XXX;

  • 可变参数列表的基本用法。参数个数和类型通过约定保证一致

    void func(int arg_list,...){
    int i,j,k;
    va_list ptr;
    i=arg_list;
    va_start(ptr, arg_list); //从参数列表中获取指针起始位置
    j=va_arg(ptr, type); //获取第二个参数，第一个参数由arg_list表示
    ... ...
    va_end(ptr); //结束获取

  • malloc 分配n字节undef空间
    calloc 分配n*size个已初始化为0的空间，效率比malloc低

  • signal的签名- 

    void (*signal (int sig, void (* handler)(int))))(int)

    分析：首先`signal是函数指针，指向的函数类型为

    默认的handler有SIG_DFL(默认行为)，SIG_IGN（忽略）
    信号有

    • SIGABRT abort中止
    • SIGFPE float process error浮点错误
    • SIGILL illegal 非法指令
    • SIGINT interrupt 中断
    • SIGSEGV segment violation 段错误

    • SIGTERM terminal 发给程序的终止消息

      int raise(int sig)用来发送信号

《C专家编程》

  • ANSI C, 标准的C标准
  • sizeof返回的是无符号值，做比较时要注意
  • #define total_cnt (sizeof(array)/sizeof(array[0]))
    可自动计算数组元素个数
  • 空指针是大写的NULL，也就是void *0
  • C语言关键字的二义性
    static 函数内部 静态变量，函数外部 文件可见性
    extern 函数定义时 全局可见性，变量/函数声明时 外部变量
    void ??
